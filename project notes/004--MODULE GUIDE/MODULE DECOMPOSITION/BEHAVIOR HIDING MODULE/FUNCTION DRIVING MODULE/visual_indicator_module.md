# Visual Indicator Module

This module manages all visual indicators, warning lights, and status displays throughout the aircraft. It controls LED indicators, warning lights, and status displays to provide clear visual feedback on system states. The module handles color coding, flashing patterns, and brightness control for different types of indicators. It coordinates with the audible signal module to provide redundant alerting and can adapt display characteristics based on ambient lighting conditions. The module includes self-test functions and can prioritize visual signals based on urgency.
